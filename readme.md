### Installation

You'll need Node.js, besides a few dependencies. 

```bash
npm install express
npm install nodemon -D
npm install nunjucks
```

### Usage
go to server.js and change the port to your preference, 3000 by. Then, run npm start in terminal.
```bash
$ npm start
```
